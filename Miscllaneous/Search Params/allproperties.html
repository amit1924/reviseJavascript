URLSearchParams Properties and Methods

1. Properties

Internally managed properties: `URLSearchParams` maintains the query parameters as an iterable collection of key-value pairs, but there are no direct properties that can be accessed like traditional properties. Instead, methods are used to interact with this internal data.

2. Methods

.append(name, value)

Adds a new value or appends to an existing key.

const params = new URLSearchParams();
params.append('key1', 'value1');
params.append('key1', 'value2');
console.log(params.toString()); // Output: "key1=value1&key1=value2"

.delete(name)

Removes a key and its values from the `URLSearchParams` object.

const params = new URLSearchParams('key1=value1&key2=value2');
params.delete('key1');
console.log(params.toString()); // Output: "key2=value2"

.get(name)

Returns the first value associated with the given key.

const params = new URLSearchParams('key1=value1&key2=value2');
console.log(params.get('key1')); // Output: "value1"

.getAll(name)

Returns all values associated with the given key as an array.

const params = new URLSearchParams('key1=value1&key1=value2');
console.log(params.getAll('key1')); // Output: ["value1", "value2"]

.has(name)

Returns `true` if the given key exists, otherwise `false`.

const params = new URLSearchParams('key1=value1');
console.log(params.has('key1')); // Output: true
console.log(params.has('key2')); // Output: false

.set(name, value)

Sets the value associated with the given key. If the key already exists, its value is updated.

const params = new URLSearchParams('key1=value1');
params.set('key1', 'newValue');
console.log(params.toString()); // Output: "key1=newValue"

.sort()

Sorts the parameters by their keys in ascending order.

const params = new URLSearchParams('b=valueB&a=valueA');
params.sort();
console.log(params.toString()); // Output: "a=valueA&b=valueB"

.toString()

Converts the `URLSearchParams` object to a query string.

const params = new URLSearchParams();
params.set('key1', 'value1');
params.set('key2', 'value2');
console.log(params.toString()); // Output: "key1=value1&key2=value2"

Example Code:

const params = new URLSearchParams('name=John&age=30');

// Append a new value to an existing key
params.append('name', 'Doe');
console.log('After append:', params.toString());

// Set a new value for an existing key
params.set('age', '31');
console.log('After set:', params.toString());

// Get a value for a key
console.log('Get name:', params.get('name'));

// Get all values for a key
console.log('Get all names:', params.getAll('name'));

// Check if a key exists
console.log('Has age:', params.has('age'));
console.log('Has gender:', params.has('gender'));

// Delete a key
params.delete('age');
console.log('After delete:', params.toString());

// Sort the parameters
params.sort();
console.log('After sort:', params.toString());
